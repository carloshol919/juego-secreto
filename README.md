# 🎮 Juego del Número Secreto

¡Bienvenido al proyecto **Número Secreto**! Este es un juego interactivo de adivinanzas desarrollado en entornos web utilizando **JavaScript, HTML y CSS**. El objetivo es desafiar al usuario a adivinar un número generado aleatoriamente por el sistema dentro de un rango predefinido, ofreciendo pistas en tiempo real.

---

## 🚀 Características

- 🎲 **Generación Aleatoria:** Cada partida genera un número diferente de forma automática.
- 💡 **Pistas en Tiempo Real:** El sistema le indica al jugador si el número secreto es *mayor* o *menor* que el número ingresado.
- ⏱️ **Contador de Intentos:** Muestra el número total de intentos que le tomó al usuario adivinar la solución al final de la partida.
- 🔄 **Rejugabilidad:** Incluye un botón para reiniciar el juego y comenzar una nueva partida sin necesidad de recargar la página.
- 🚫 **Validación de Datos:** Evita repetir números ya ganados en la sesión actual (opcional según la versión).

---

## 🛠️ Tecnologías Utilizadas

Para el desarrollo de este juego se emplearon las siguientes tecnologías:

- **HTML5:** Estructuración del contenido y elementos visuales del juego.
- **CSS3:** Diseño responsivo, estilos personalizados y una interfaz de usuario atractiva.
- **JavaScript (ES6+):** Manipulación del DOM, lógica de control, condicionales y funciones matemáticas para el funcionamiento interno del juego.

🕹️ Cómo Jugar
1. Al iniciar el juego, el sistema elegirá un número secreto (por ejemplo, del 1 al 100).

2. Introduce un número en el campo de texto de la pantalla.

3. Haz clic en el botón "Intentar".

4. Lee la pista en pantalla:

 - Si fallas, el juego te dirá si el número es mayor o menor.

  -Si aciertas, ¡felicidades! El juego te mostrará cuántos intentos te tomó ganar.

5. Presiona el botón "Nuevo Juego" para reiniciar el desafío.
